## 📋 ЗАДАЧИ ДЛЯ ЭКСПЕРТОВ (ОБНОВЛЕНО)

### 1. Маркетолог (Lead)
- Координация 12 каналов
- Создание уникальных `start_param` для каждого знака:
  - `tiktok_aries1`, `tiktok_taurus1`, ... `tiktok_pisces1`
  - `yt_aries1`, `yt_taurus1`, ... `yt_pisces1`
- Мониторинг эффективности каждого знака (кто дает больше конверсии)
- Выявление "золотых" знаков (где лучшая конверсия) и перераспределение ресурсов

### 2. Контент-мейкер (Video)
- **Съемка 24 видео в день** (по 2 на каждый знак)
- Адаптация скриптов под специфику знака:
  - Овны: "Овны, сегодня идеальный день для старта!"
  - Рыбы: "Рыбы, прислушайтесь к интуиции..."
- Использование специфических хештегов: `#овны #aries #гороскоповны`

### 3. Креатор (Design/Copywriting)
- Создание **12 уникальных аватарок** (по знакам зодиака)
- 12 вариантов био (адаптированных под знак)
- Шаблоны видео с символикой знака (цвета, шрифты, эмодзи)
- Пример био для Овна:
  ```
  ♈ Овен: твой гороскоп на сегодня
  ✨ Энергия, страсть, драйв
  👉 Ссылка в профиле!
  ```

### 4. Community Manager
- Ведение 12 отдельных профилей (ответы на комментарии)
- Создание 12 чатов/каналов в Telegram (по знакам) для удержания
- Работа с лояльностью "своих" пользователей

### 5. TikTok Specialist (КРИТИЧЕСКИЙ)
- **Проверка 24 видео в день** (по 2 на каждый знак)
- Контроль отсутствия прямых ссылок в тексте (только в Website URL)
- Управление прокси-аккаунтами (по 1-2 на знак)
- Мониторинг банов и оперативная замена упавших аккаунтов
- **Главное правило**: 12 каналов = 12 разных устройств/IP. Не логинить все с одного IP!

### 6. YouTube Safety Specialist (КРИТИЧЕСКИЙ)
- **Проверка 24 Shorts в день** (по 2 на каждый знак для YouTube)
- Контроль безопасности: ссылки ТОЛЬКО в конце описания или закрепленном комментарии
- Музыка ТОЛЬКО из YouTube Audio Library (бесплатно, без авторских прав)
- Проверка метаданных (заголовок до 60 символов, 5-7 тегов, без спама)
- Мониторинг демонетизации и шадоу-банов (ограничение охватов)
- **Главное правило**: Первые 30 сек видео — НЕТ ссылок, НЕТ призывов "переходи"
- Ссылки для трафика: `?start=yt_aries1`, `?start=yt_taurus1`... (уникальные для YouTube)
