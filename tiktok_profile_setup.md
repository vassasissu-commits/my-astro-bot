## 4. Ссылка в био (Website URL) — ГДЕ НАХОДИТСЯ
*Единственное место, где можно оставить прямую ссылку. **Используй только здесь**.*

### Как найти поле Website URL:
1. Открой **TikTok** → нажми **«Я» (Profile)** внизу справа
2. Нажми **«Edit profile» (Редактировать профиль)**
3. Листай вниз — там есть поле **«Website»** или **«URL»** (под полем Bio)
4. Вставь туда ссылку: `https://t.me/SuperAstro2027_bot?start=tiktok1`
5. Нажми **«Save» (Сохранить)**

### Структура профиля:
```
Profile
├── Name: 🔮 Vedana
├── Username: @vedana_astro
├── Bio: ✨ Поддержка и прогноз на каждый день 👇  ← ТУТ НЕТ ССЫЛКИ!
└── Website: [сюда вставляешь ссылку]  ← ССЫЛКА ТОЛЬКО ЗДЕСЬ!
```

**Правильный формат ссылки**:
```
https://t.me/SuperAstro2027_bot?start=tiktok1
```

**Важно**: 
- Добавь `?start=tiktok1` для отслеживания источника в логах бота
- В поле Website ссылка кликабельная — пользователи нажимают и попадают в Telegram
- В тексте Bio (био) ссылок НЕТ — только "ссылка в профиле" / стрелочка 👇

## 6. Чек-лист перед публикацией (TikTok Specialist)
Перед тем как заполнить профиль и начать постить, проверь:

- [ ] В имени нет слова "бот", "Telegram", "tg"
- [ ] В био нет `t.me`, `http`, `https`
- [ ] В био есть мягкий призыв "Смотри в профиле" / "Ссылка в био"
- [ ] В поле Website URL стоит правильная ссылка с `start_param`
- [ ] Аватарка без текста и ссылок
- [ ] Аккаунт не привязан к телефону, с которого были баны

## 7. Пошаговая инструкция заполнения
1. Зайди в настройки профиля TikTok
2. Нажми "Редактировать профиль"
3. Вставь **Имя** (вариант A)
4. Вставь **Username** (вариант A)
5. Вставь **Bio** (вариант A)
6. Вставь **Website URL** (`https://t.me/SuperAstro2027_bot?start=tiktok1`)
7. Загрузи **Аватарку** (без текста)
8. Сохрани и **НЕ ПУБЛИКУЙ** видео сразу — дай TikTok Specialist проверить.
